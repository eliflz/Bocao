# 🍽️ Bocao

Bienvenido a **Bocao** 🚀, un proyecto privado que busca crear una **aplicación web para un restaurante**.  
El objetivo es ofrecer una interfaz moderna, intuitiva y atractiva para los usuarios.  

---

## ✨ Características principales

- 🏪 **Aplicación web para restaurante**  
- 🎨 **Interfaz de usuario** desarrollada con **HTML** y **CSS**  
- ⚡ Diseño simple, ligero y fácil de personalizar  

---

## 🛠️ Pila Tecnológica

- 🌐 **HTML5** – Estructura y contenido  
- 🎨 **CSS3** – Estilos y diseño visual
