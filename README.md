# 🍔 Bocao – Web App para Restaurante

**Bocao** es una aplicación web moderna para restaurantes que permite a los usuarios **explorar el menú, agregar productos a un carrito y realizar pedidos de forma interactiva**.

El proyecto está diseñado con un enfoque en **experiencia de usuario, diseño moderno y simplicidad**, utilizando tecnologías web básicas para lograr una interfaz rápida, ligera y visualmente atractiva.

---

# 🚀 Características

### 🍽️ Menú interactivo
Los usuarios pueden explorar el menú del restaurante y visualizar:

- Nombre del producto
- Descripción
- Precio
- Imagen del producto

Cada producto puede **agregarse al carrito con un solo clic**.

---

### 🛒 Carrito de compras dinámico

El sistema incluye un carrito funcional que permite:

- Agregar productos
- Aumentar la cantidad automáticamente
- Eliminar productos del carrito
- Ver el total del pedido
- Simular la confirmación del pedido

Toda la lógica del carrito está desarrollada con **JavaScript del lado del cliente**.

---

### 🎨 Interfaz moderna

La aplicación cuenta con:

- Animaciones suaves
- Tarjetas de productos
- Modal para el carrito
- Efectos hover
- Elementos flotantes en la interfaz

El diseño fue construido utilizando **CSS moderno con variables, animaciones y grid layouts**.

---

### 📱 Diseño Responsive

La interfaz se adapta automáticamente a distintos dispositivos:

- Computadores
- Tablets
- Teléfonos móviles

Gracias al uso de **CSS Grid y Media Queries**.

---

# 🛠️ Tecnologías Utilizadas

- **HTML5** – Estructura de la aplicación  
- **CSS3** – Estilos, animaciones y diseño responsive  
- **JavaScript (Vanilla)** – Lógica del carrito y eventos interactivos
